## Firewall

A firewall is a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.

## What is a Firewall?

A firewall acts as a barrier between trusted and untrusted networks, filtering traffic according to a defined ruleset.

**Types**:
- **Packet Filtering**: Basic IP/port filtering
- **Stateful**: Tracks connection state
- **Application Layer**: Deep packet inspection
- **Next-Gen**: Combines multiple techniques

**Common Rules**:
- Allow/Deny by IP address
- Allow/Deny by port
- Allow/Deny by protocol
- Default deny policy

## Features

This implementation provides:
- Rule-based filtering
- IP/CIDR matching
- Port and protocol filtering
- Default policies
